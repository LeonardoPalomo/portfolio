Mi rol en este proyecto consistió en desarrollar interfaces de usuario para monitorear el estado de trincheras de composts ubicados en otra región. Cada trinchera estaba equipada con sensores capaces de registrar temperatura, nivel de batería y estado de conectividad. Esta información era enviada mediante Amazon Web Services (AWS), donde era procesada para posteriormente ser presentada a los usuarios.

Los datos se visualizaron a través de dos interfaces principales: una vista resumida que mostraba el estado general de todas las zanjas monitoreadas y una vista detallada que permitía inspeccionar la información específica de cada una de ellas.

Demostración:

Al iniciar sesión, el usuario accede a una interfaz que muestra todas las trincheras junto con información resumida, como por ejemplo la última temperatura registrada. Desde esta vista es posible seleccionar cualquier trinchera para acceder a un panel detallado, donde se presenta información histórica sobre temperatura y niveles de batería dentro de un rango de fechas configurable.

Además, se incorporó una funcionalidad de exportación que permite guardar el gráfico exactamente como se visualiza en pantalla, incluyendo todos los datos y filtros seleccionados por el usuario.
